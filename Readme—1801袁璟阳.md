## 游戏规则
当运行游戏时候，画面静止不动，可以默认让蛇头超右，游戏中设置w s a d 4个按键分别代表，上，下，左，右，也是用户比较常用的方向按键，当用户输入w或者s或者d时候激活游戏，注意输入a不可以激活，因为蛇不可以180°转弯，因此蛇的移动方向只可以一直向前或者90°旋转。
当蛇吃掉食物时候，此时蛇会增加一个身段，另外食物需要重新随机的设置到屏幕上。
